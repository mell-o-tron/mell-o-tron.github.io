## (Ultra)Filter

A **filter** on $\mathcal P(I)$ is a subset $F \subseteq \mathcal P(I)$ such that:

1. $\emptyset \notin F$, $I \in F$
2. $A \in F$, $A \subseteq B \subseteq I \implies B \in F$
3. $A, B \in F \implies A \cap B \in F$

A filter is called an **ultrafilter** if:

4. $A \cup B = I \implies A \in U \vee B \in U$.

### Principal ultrafilters

The set $U = \{A \subseteq I \mid x \in A\}$ for $x \in I$ is called the *principal ultrafilter generated by* $x$. Usually, we assume that the ultrafilters considered are not principal.

### Cofinite filter

Given a set $I$, the set of all its cofinite subsets is a filter on $\mathcal P(I)$. It is not an ultrafilter (e.g., partition the naturals into even and odd sets when $I = \mathbb N$).
